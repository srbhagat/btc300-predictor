# BTCUSDT 300-Point Cloud Predictor

Cloud-safe version.

## Why this version exists
Streamlit Cloud can return HTTP 451 when calling Binance Futures API directly.
This version uses Binance public monthly archive files from data.binance.vision instead.

## Files
- app.py
- requirements.txt
- README.md

## Deploy
Upload these files to GitHub and deploy app.py on Streamlit Cloud.
