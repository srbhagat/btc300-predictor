# BTCUSDT 300-Point Fast Accurate India

Fast + accurate decision-support app for BTCUSDT 300-point trades.

## Optimized for
- India timezone
- iPhone web app use
- Fast refresh
- 5-year data
- No manual button clicks

## Run
```cmd
pip install -r requirements.txt
streamlit run app.py
```

## Reuse existing data
Copy the `data` folder from your earlier app into this app folder.
