# BTCUSDT 300-Point CoinDCX Only Predictor

This version uses only CoinDCX public market data.

## Data sources
- CoinDCX public candles endpoint
- CoinDCX futures live trade endpoint

## No Binance
This app does not use Binance data or Binance APIs.

## Deploy
Upload app.py, requirements.txt, and README.md to GitHub and reboot Streamlit Cloud.
